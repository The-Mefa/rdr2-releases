## 🚀 Kullanım

1. Kurulum dosyasını indir ve çalıştır
2. **Devam Et** butonuna bas
3. RDR2 oyun klasörünü seç
4. Önce **ScriptHookRDR2**'yi kur
5. İstediğin diğer modları kur
